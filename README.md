# Stopwatch Web Application

## Features
- Start Stopwatch
- Pause Stopwatch
- Reset Stopwatch
- Record Lap Times
- Express.js Backend
- Responsive UI

## Technologies Used
- HTML
- CSS
- JavaScript
- Node.js
- Express.js

## How to Run

1. Install dependencies:
   npm install

2. Start the server:
   node server.js

3. Open:
   http://localhost:3000
